# context/

Papers for when input context is too long, noisy, missing information, or poorly framed.

## Subdirectories

| Directory       | What                                     | When to read                         |
| --------------- | ---------------------------------------- | ------------------------------------ |
| `reframing/`    | Restructure context for better attention | Context is noisy, poorly framed      |
| `augmentation/` | Retrieve and inject missing information  | Model lacks knowledge to reason with |
