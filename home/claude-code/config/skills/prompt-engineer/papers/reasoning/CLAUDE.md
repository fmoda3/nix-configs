# reasoning/

Papers for when model fails to work through complex problems systematically.

## Subdirectories

| Directory        | What                             | When to read                             |
| ---------------- | -------------------------------- | ---------------------------------------- |
| `decomposition/` | Break problems into sub-problems | Model can't handle multi-step complexity |
| `elicitation/`   | Draw out step-by-step reasoning  | Model skips steps, doesn't show work     |
